## LEARNING

### Create New Template

Just copy the readbase folder and make it yours.

```sh
cp -r "source/readbase" "source/<template>"
```

### Trigger Template Capture

You can run the update templates workflow manually.

```sh
gh workflow run "op-update-templates.yml"
```
